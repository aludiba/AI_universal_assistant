## 📝 运行说明

### 1. 安装依赖
```bash
cd AIUniversalAssistantFlutter
flutter pub get
```

### 2. 配置
- 在 `lib/config/app_config.dart` 中配置API Key
- 确保 `assets/data/` 目录下的JSON文件存在

### 3. 运行
```bash
flutter run -d ios      # iOS
flutter run -d android  # Android
```
