## 📝 使用说明

### 1. 安装依赖
```bash
cd AIUniversalAssistantFlutter
flutter pub get
```

### 2. 配置API Key
在 `lib/config/app_config.dart` 中配置：
- DeepSeek API Key
- IAP产品ID（如需要）

### 3. 运行项目
```bash
flutter run -d ios      # iOS
flutter run -d android  # Android
```
