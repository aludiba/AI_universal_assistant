## 🚀 部署步骤

### 方案一：使用 FTP/SFTP 客户端（最简单）

1. **打开 FTP 客户端**（推荐 FileZilla、WinSCP）
2. **连接到服务器**：
   - 主机：您的服务器 IP 或域名
   - 用户名：服务器用户名
   - 密码：服务器密码
   - 端口：22（SFTP）或 21（FTP）
3. **上传文件**：
   - 将 `website-deploy/` 目录下的所有文件上传到网站根目录
   - 通常是 `/var/www/html/` 或 `/public_html/` 或 `/www/`
4. **设置文件权限**：
   - HTML 文件：644
   - 目录：755

### 方案二：使用命令行（SSH）

```bash
# 进入部署目录
cd /Users/chb/Desktop/AI_universal_assistant/AIUniversalAssistant/website-deploy

# 使用 scp 上传
scp -r * user@your-server:/var/www/html/

# 或使用 rsync（推荐）
rsync -avz --progress * user@your-server:/var/www/html/
```

### 方案三：使用部署脚本

```bash
cd website-deploy
./deploy.sh your-server.com username /var/www/html
```

## 🌐 DNS 配置

确保域名 `hujiaofenwritingcat.top` 已正确解析：

**如果使用自己的服务器：**
- 类型：A 记录
- 名称：@（根域名）或 www
- 值：服务器 IP 地址

**如果使用 GitHub Pages/Netlify/Vercel：**
- 类型：CNAME 记录
- 名称：@ 或 www
- 值：提供的 CNAME 目标

## 🔧 常见问题

**Q: 访问网站显示 404？**
A: 检查文件是否上传到正确目录，确认首页文件名为 `index.html`

**Q: 图片无法显示？**
A: 检查 `images/` 目录是否已上传，确认图片路径为相对路径

**Q: 链接无法访问？**
A: 确认文件权限正确（644），检查服务器配置

**Q: 域名无法访问？**
A: 检查 DNS 解析是否生效，可使用 `nslookup hujiaofenwritingcat.top` 检查
