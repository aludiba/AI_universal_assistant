# 穿山甲SDK版本更新说明

## 当前版本：Ads-CN 6.7.0.8

### ✅ 已修复的编译错误

#### 1. 移除 `appName` 属性
**错误信息**：
```
Property 'appName' not found on object of type 'BUAdSDKConfiguration *'
```

**原因**：SDK 6.7+ 版本简化了配置，移除了 `appName` 属性

**修复方案**：
```objective-c
// ❌ 旧版本代码（6.3及以下）
configuration.appName = @"AI Universal Assistant";

// ✅ 新版本代码（6.7+）
// 不再需要设置 appName
```

#### 2. 移除 `logLevel` 属性
**错误信息**：
```
Property 'logLevel' not found on object of type 'BUAdSDKConfiguration *'
Use of undeclared identifier 'BUAdSDKLogLevelDebug'
```

**原因**：SDK 6.7+ 版本移除了手动设置日志级别的功能

**修复方案**：
```objective-c
// ❌ 旧版本代码（6.3及以下）
configuration.logLevel = BUAdSDKLogLevelDebug;

// ✅ 新版本代码（6.7+）
// 不再需要设置 logLevel，日志会自动输出到Xcode控制台
```

### 📝 完整的初始化代码对比

#### 旧版本（SDK 6.3）
```objective-c
- (void)initPangleSDK {
    NSString *appID = AIUA_APPID;
    
    BUAdSDKConfiguration *configuration = [BUAdSDKConfiguration configuration];
    configuration.appID = appID;
    configuration.appName = @"AI Universal Assistant";  // ❌ 已移除
    configuration.logLevel = BUAdSDKLogLevelDebug;      // ❌ 已移除
    
    [BUAdSDKManager startWithAsyncCompletionHandler:^(BOOL success, NSError *error) {
        // 回调处理
    }];
}
```

#### 新版本（SDK 6.7+）
```objective-c
- (void)initPangleSDK {
    NSString *appID = AIUA_APPID;
    
    // 创建配置（SDK 6.7+ 版本简化了配置，只需设置AppID）
    BUAdSDKConfiguration *configuration = [BUAdSDKConfiguration configuration];
    configuration.appID = appID;  // ✅ 只需设置 AppID
    
    // 注意：新版本SDK已移除 appName 和 logLevel 属性
    // 如需调试日志，请在Xcode中查看控制台输出
    
    [BUAdSDKManager startWithAsyncCompletionHandler:^(BOOL success, NSError *error) {
        // 回调处理
    }];
}
```

### 🔄 API变更总结

| 属性/方法 | 旧版本(6.3) | 新版本(6.7+) | 说明 |
|---------|-----------|------------|------|
| `appID` | ✅ 必填 | ✅ 必填 | 无变化 |
| `appName` | ✅ 可选 | ❌ 已移除 | SDK内部自动获取 |
| `logLevel` | ✅ 可选 | ❌ 已移除 | 自动输出日志 |
| `startWithAsyncCompletionHandler:` | ✅ | ✅ | 无变化 |

### 💡 迁移建议

如果你从旧版本升级到新版本：

1. **删除 `appName` 设置**
   ```objective-c
   // 删除这一行
   // configuration.appName = @"XXX";
   ```

2. **删除 `logLevel` 设置**
   ```objective-c
   // 删除这一行
   // configuration.logLevel = BUAdSDKLogLevelDebug;
   ```

3. **保持其他代码不变**
   - AppID 设置方式不变
   - 初始化方法不变
   - 回调处理不变
