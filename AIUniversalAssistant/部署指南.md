# AI创作喵网站部署指南

本指南将帮助您将网站部署到域名 `hujiaofenwritingcat.top`。

## 📁 网站文件结构

部署时需要上传以下文件到服务器：

```
网站根目录/
├── index.html              # 官网首页（从 AI创作喵-官网.html 重命名）
├── user-agreement.html     # 用户协议（从 用户协议.html 重命名）
├── privacy-policy.html     # 隐私政策（从 隐私政策.html 重命名）
└── images/                 # 图片资源目录
    ├── screenshot1.png
    ├── screenshot2.png
    └── screenshot3.png
```

## 🚀 部署方案

### 方案一：使用 FTP/SFTP 上传到服务器（推荐）

如果您已经有服务器和域名解析配置：

1. **准备文件**
   - 将 `AI创作喵-官网.html` 重命名为 `index.html`
   - 将 `用户协议.html` 重命名为 `user-agreement.html`
   - 将 `隐私政策.html` 重命名为 `privacy-policy.html`
   - 确保 `images/` 目录包含所有截图

2. **使用 FTP 客户端上传**
   - 使用 FileZilla、WinSCP 或其他 FTP 客户端
   - 连接到您的服务器
   - 上传所有文件到网站根目录（通常是 `public_html/` 或 `www/` 目录）

3. **配置域名**
   - 确保域名 `hujiaofenwritingcat.top` 已解析到服务器 IP
   - 在服务器上配置虚拟主机指向网站根目录

### 方案二：使用 GitHub Pages（免费）

1. **创建 GitHub 仓库**
   ```bash
   # 在项目目录下执行
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/yourusername/ai-writing-cat-website.git
   git push -u origin main
   ```

2. **启用 GitHub Pages**
   - 进入仓库 Settings → Pages
   - Source 选择 `main` 分支，`/root` 目录
   - 保存后访问 `https://yourusername.github.io/ai-writing-cat-website/`

3. **配置自定义域名**
   - 在仓库根目录创建 `CNAME` 文件，内容：`hujiaofenwritingcat.top`
   - 在域名 DNS 中添加 CNAME 记录：
     - 类型：CNAME
     - 名称：@（或 www）
     - 值：yourusername.github.io

### 方案三：使用 Netlify（免费，推荐）

1. **注册 Netlify 账号**
   - 访问 https://www.netlify.com
   - 使用 GitHub 账号登录

2. **部署网站**
   - 点击 "Add new site" → "Deploy manually"
   - 将网站文件拖拽上传，或使用 Netlify CLI：
     ```bash
     npm install -g netlify-cli
     netlify deploy --prod
     ```

3. **配置自定义域名**
   - 在 Netlify 控制台：Site settings → Domain management
   - 添加自定义域名：`hujiaofenwritingcat.top`
   - 按照提示配置 DNS 记录

### 方案四：使用 Vercel（免费）

1. **安装 Vercel CLI**
   ```bash
   npm install -g vercel
   ```

2. **部署**
   ```bash
   cd 网站目录
   vercel
   ```

3. **配置域名**
   - 在 Vercel 控制台添加自定义域名
   - 配置 DNS 记录

## 🌐 DNS 配置

无论使用哪种方案，都需要配置 DNS 记录：

**如果使用 GitHub Pages/Netlify/Vercel：**
- 类型：CNAME
- 名称：@（根域名）或 www
- 值：提供的 CNAME 目标

**如果使用自己的服务器：**
- 类型：A
- 名称：@（根域名）
- 值：服务器 IP 地址

## 🆘 常见问题

**Q: 访问网站显示 404？**
A: 检查文件是否上传到正确目录，确保首页文件名为 `index.html`

**Q: 图片无法显示？**
A: 检查图片路径是否正确，确保 `images/` 目录已上传

**Q: 域名无法访问？**
A: 检查 DNS 解析是否生效，可使用 `nslookup hujiaofenwritingcat.top` 检查
